# HarryPotterQuotesCLI

A Command Line Interface which fetches random harry potter quotes.

## Installation

```
npm i harrypotter-quotes
```

### Help 

Help for knowing commands

```
quotes help
```

### To start quotes

```
quotes start -- quotes
or ,
quotes start --q
```


### To continue with next quotes

```
quotes next -- quotes
or ,
quotes next --q
```
